Linux (o GNU/Linux, para complacer a RMS) suele clasificarse como uno de los sistemas operativos mas personalizables y es famoso por la gran libertad que se le da al usuario

Pero como algunos podran saber, Linux por si solo no es un sistema operativo, sino el kernel sobre el cual el SO funciona. Para poder considerarse un sistema operativo, suele distribuirse el kernel, y ademas, paquetes adicionales, como por ejemplo un Entorno de Escritorio, un gestor de paquetes, entre otros paquetes que componen el sistema, como las librerias GNU

A esto se le llama Distribucion Linux. Actualmente existen muchas distribuciones Linux, las cuales pueden tener distintos fines y pueden estar destinadas a distintos tipos de usuarios

Por ejemplo, tenemos distribuciones que estan diseñadas especialmente para usuarios poco experimentados, las cuales vienen completamente pre-configuradas para hacer que la experiencia sea lo mas facil y estable posible. Entre este tipo de distribuciones podemos encontrar distribuciones como:

  • Ubuntu
  • Pop!_OS
  • Deepin
  • Linux Mint
  • etc.
